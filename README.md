# ProctoSphere

ProctoSphere is a secure proctored exam platform designed for shortlisting candidates for national and international level hackathons. It provides standardized online assessments, real-time proctoring, violation tracking, automated scoring, and candidate shortlisting.

## Features

### Admin
- Secure admin dashboard
- Create and manage exams
- Add MCQ questions
- View candidate attempts
- View proctoring logs
- Leaderboard and analytics
- Shortlisted/rejected candidate tracking

### Candidate
- Register and login
- View available exams
- Instruction page before exam
- Webcam permission and preview
- Attempt timed exams
- View result and status

### Proctoring
- Webcam access verification
- Tab switching detection
- Copy/paste blocking
- Right-click blocking
- Fullscreen monitoring
- Violation logging
- Violation-based score penalty
- Multiple attempt prevention

## Tech Stack

- Laravel
- PHP
- MySQL
- Blade
- Tailwind CSS
- JavaScript
- Laravel Breeze
- Vite

## Project Purpose

Hackathon organizers often struggle with evaluating candidates fairly and efficiently. ProctoSphere solves this problem by providing a standardized and secure assessment system for candidate screening.
